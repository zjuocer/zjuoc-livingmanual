# 短学期

海洋学院有着较长的短学期。短学期的通知可在学院办公网搜索到。

- [2026年暑假短学期](https://oca.zju.edu.cn/2026/0519/c29988a3164289/page.psp)
- [2025年暑假短学期](https://oca.zju.edu.cn/2025/0527/c29988a3055686/page.psp)

按照惯例，海洋学院的短学期开始于校历的假期第一天。以下列出近几年短学期的开始日期：

- 2026年暑假：7月6日
- 2025年暑假：6月25日

大一的短学期均在舟山进行，分别为海洋科学专业的`海洋科学概论`和海洋工程与技术专业的`海洋工程与技术认知实习`。24级、25级的时长为21天。

大二的海洋资源与能源方向的短学期在杭州进行，为21天，其他方向短学期均在舟山进行，为14天。
